Математический анализ: Определения, Формулы, Теоремы и Примеры
1. Основные понятия
Определение 1.1: Последовательность
Последовательность чисел — это функция, определенная на множестве натуральных чисел. Обозначается как {a_n} или a₁, a₂, a₃, ..., где n ∈ ℕ. Каждому натуральному числу n соответствует единственное число a_n из множества действительных или комплексных чисел.

Определение 1.2: Предел последовательности
Число L называется пределом последовательности {a_n}, если для любого положительного числа ε, существует натуральное число N такое, что для всех n > N выполняется неравенство |a_n - L| < ε. Обозначается как: lim(n→∞) a_n = L.

Определение 1.3: Функция
Функция f : D → R есть соответствие, при котором каждому элементу x из множества D (область определения) соответствует ровно один элемент y из множества R (область значения). Пишется y = f(x).

Определение 1.4: Непрерывность функции
Функция f(x) называется непрерывной в точке x₀, если выполняются три условия: (1) f(x₀) определена; (2) существует lim(x→x₀) f(x); (3) lim(x→x→x₀) f(x) = f(x₀).

2. Пределы и непрерывность
Теорема 2.1: Единственность предела
Если последовательность {a_n} имеет предел, то этот предел единственен.

Теорема 2.2: Арифметические операции с пределами
Если lim(n→∞) a_n = A и lim(n→∞) b_n = B, то:

lim(n→∞) (a_n + b_n) = A + B

lim(n→∞) (a_n - b_n) = A - B

lim(n→∞) (a_n · b_n) = A · B

lim(n→∞) (a_n / b_n) = A / B, при условии B ≠ 0

Пример 2.1: Вычисление предела последовательности
Найти lim(n→∞) (3n² + 2n) / (n² + 1).

Решение: Разделим числитель и знаменатель на n²:
lim(n→∞) (3n² + 2n) / (n² + 1) = lim(n→∞) (3 + 2/n) / (1 + 1/n²) = (3 + 0) / (1 + 0) = 3.

Теорема 2.3: Теорема о сжатой переменной (Squeeze Theorem)
Если для всех n > N выполняется a_n ≤ c_n ≤ b_n и lim(n→∞) a_n = lim(n→∞) b_n = L, то lim(n→∞) c_n = L.

Пример 2.2: Применение теоремы о сжатой переменной
Найти lim(n→∞) sin(n) / n.

Решение: Известно, что -1 ≤ sin(n) ≤ 1 для всех n. Следовательно:
-1/n ≤ sin(n)/n ≤ 1/n.
Так как lim(n→∞) (-1/n) = 0 и lim(n→∞) (1/n) = 0, по теореме о сжатой переменной:
lim(n→∞) sin(n) / n = 0.

3. Производная и дифференцирование
Определение 3.1: Производная функции
Производная функции f(x) в точке x₀ определяется как предел:
f'(x₀) = lim(h→0) [f(x₀ + h) - f(x₀)] / h, если этот предел существует.

Альтернативная форма: f'(x₀) = lim(x→x₀) [f(x) - f(x₀)] / (x - x₀).

Определение 3.2: Дифференцируемость
Функция называется дифференцируемой на интервале, если она имеет производную в каждой точке этого интервала.

Теорема 3.1: Дифференцируемость влечет непрерывность
Если функция f(x) дифференцируема в точке x₀, то она непрерывна в точке x₀.

Правила дифференцирования:
Производная суммы: (u + v)' = u' + v'

Производная произведения: (u · v)' = u' · v + u · v'

Производная частного: (u / v)' = (u'v - uv') / v²

Правило цепи: (f(g(x)))' = f'(g(x)) · g'(x)

Таблица основных производных:
(c)' = 0, где c — константа

(xⁿ)' = n·xⁿ⁻¹

(eˣ)' = eˣ

(aˣ)' = aˣ · ln(a)

(ln(x))' = 1/x

(log_a(x))' = 1 / (x · ln(a))

(sin(x))' = cos(x)

(cos(x))' = -sin(x)

(tan(x))' = sec²(x) = 1/cos²(x)

(arcsin(x))' = 1 / √(1 - x²)

(arccos(x))' = -1 / √(1 - x²)

(arctan(x))' = 1 / (1 + x²)

Пример 3.1: Вычисление производной
Найти производную функции f(x) = 3x⁴ + 2x² - 5x + 7.

Решение:
f'(x) = 3·4x³ + 2·2x - 5 + 0 = 12x³ + 4x - 5.

Пример 3.2: Применение правила цепи
Найти производную функции f(x) = sin(2x³).

Решение: Обозначим u = 2x³, тогда f(x) = sin(u).
f'(x) = cos(2x³) · (2x³)' = cos(2x³) · 6x² = 6x² · cos(2x³).

Пример 3.3: Применение правила произведения
Найти производную функции f(x) = x² · eˣ.

Решение: Применим правило произведения:
f'(x) = (x²)' · eˣ + x² · (eˣ)' = 2x · eˣ + x² · eˣ = eˣ(2x + x²) = x·eˣ(2 + x).

4. Интегралы и интегрирование
Определение 4.1: Первообразная
Функция F(x) называется первообразной для функции f(x) на интервале (a, b), если F'(x) = f(x) для всех x ∈ (a, b).

Определение 4.2: Неопределенный интеграл
Неопределенный интеграл функции f(x) — это множество всех первообразных функции f(x). Обозначается:
∫ f(x) dx = F(x) + C, где C — произвольная постоянная интегрирования.

Определение 4.3: Определенный интеграл (интеграл Римана)
Определенный интеграл функции f(x) на отрезке [a, b] определяется как:
∫_a^b f(x) dx = F(b) - F(a), где F(x) — первообразная функции f(x).

Теорема 4.1: Основная теорема анализа
Если функция f(x) непрерывна на отрезке [a, b] и F(x) — произвольная первообразная f(x) на [a, b], то:
∫_a^b f(x) dx = F(b) - F(a).

Таблица основных интегралов:
∫ 0 dx = C

∫ xⁿ dx = xⁿ⁺¹ / (n + 1) + C, при n ≠ -1

∫ 1/x dx = ln|x| + C

∫ eˣ dx = eˣ + C

∫ aˣ dx = aˣ / ln(a) + C

∫ sin(x) dx = -cos(x) + C

∫ cos(x) dx = sin(x) + C

∫ 1/cos²(x) dx = tan(x) + C

∫ 1/√(1 - x²) dx = arcsin(x) + C

∫ 1/(1 + x²) dx = arctan(x) + C

Пример 4.1: Вычисление неопределенного интеграла
Найти ∫ (4x³ + 3x - 2) dx.

Решение:
∫ (4x³ + 3x - 2) dx = 4 · x⁴/4 + 3 · x²/2 - 2x + C = x⁴ + (3/2)x² - 2x + C.

Пример 4.2: Вычисление определенного интеграла
Найти ∫_0^2 x² dx.

Решение: Первообразная функции f(x) = x² есть F(x) = x³/3.
∫_0^2 x² dx = [x³/3]_0^2 = 2³/3 - 0³/3 = 8/3.

Правило интегрирования по частям:
∫ u dv = uv - ∫ v du.

Пример 4.3: Интегрирование по частям
Найти ∫ x·eˣ dx.

Решение: Обозначим u = x, dv = eˣ dx. Тогда du = dx, v = eˣ.
∫ x·eˣ dx = x·eˣ - ∫ eˣ dx = x·eˣ - eˣ + C = eˣ(x - 1) + C.

5. Ряды
Определение 5.1: Числовой ряд
Числовой ряд — это выражение вида:
a₁ + a₂ + a₃ + ... = Σ(n=1 to ∞) a_n, где {a_n} — последовательность чисел.

Определение 5.2: Частичная сумма ряда
Частичной суммой ряда Σ(n=1 to ∞) a_n называется сумма:
S_N = Σ(n=1 to N) a_n = a₁ + a₂ + ... + a_N.

Определение 5.3: Сходимость ряда
Ряд Σ(n=1 to ∞) a_n называется сходящимся, если существует конечный предел его частичных сумм:
lim(N→∞) S_N = S. Число S называется суммой ряда.

Теорема 5.1: Необходимое условие сходимости
Если ряд Σ(n=1 to ∞) a_n сходится, то lim(n→∞) a_n = 0.

Теорема 5.2: Критерий Коши для рядов
Ряд Σ(n=1 to ∞) a_n сходится тогда и только тогда, когда для любого ε > 0 существует натуральное число N такое, что для всех n > N и всех натуральных p выполняется:
|a_{n+1} + a_{n+2} + ... + a_{n+p}| < ε.

Пример 5.1: Геометрический ряд
Геометрический ряд имеет вид: Σ(n=0 to ∞) r^n = 1 + r + r² + r³ + ...
Этот ряд сходится при |r| < 1 и его сумма равна S = 1 / (1 - r).
При |r| ≥ 1 ряд расходится.

Пример: Σ(n=0 to ∞) (1/2)^n = 1 / (1 - 1/2) = 2.

Пример 5.2: Гармонический ряд
Гармонический ряд: Σ(n=1 to ∞) 1/n = 1 + 1/2 + 1/3 + 1/4 + ...
Этот ряд расходится, хотя lim(n→∞) 1/n = 0.

Теорема 5.3: Признак сравнения
Если 0 ≤ a_n ≤ b_n для всех n и ряд Σ b_n сходится, то ряд Σ a_n сходится.
Если a_n ≥ b_n ≥ 0 и ряд Σ b_n расходится, то ряд Σ a_n расходится.

Теорема 5.4: Признак Д'Аламбера
Пусть для ряда Σ(n=1 to ∞) a_n с положительными членами существует предел:
l = lim(n→∞) a_{n+1} / a_n.
Если l < 1, ряд сходится. Если l > 1, ряд расходится. Если l = 1, тест неопределен.

6. Функции нескольких переменных
Определение 6.1: Функция двух переменных
Функция двух переменных z = f(x, y) — это соответствие, которое каждой паре (x, y) из области определения D ⊂ ℝ² ставит в соответствие единственное вещественное число z ∈ ℝ.

Определение 6.2: Частная производная
Частная производная функции z = f(x, y) по переменной x в точке (x₀, y₀) определяется как:
∂f/∂x|(x₀,y₀) = lim(h→0) [f(x₀ + h, y₀) - f(x₀, y₀)] / h.

Теорема 6.1: Смешанные частные производные
Если функция f(x, y) имеет непрерывные частные производные второго порядка, то смешанные частные производные равны:
∂²f/∂x∂y = ∂²f/∂y∂x.

Пример 6.1: Вычисление частных производных
Найти ∂f/∂x и ∂f/∂y для функции f(x, y) = x²y + xy² + x + y.

Решение:
∂f/∂x = 2xy + y² + 1
∂f/∂y = x² + 2xy + 1
